This is a Smalltalk-80 implementation written in Micron, following the famous 
[Smalltalk "Bluebook"](http://stephane.ducasse.free.fr/FreeBooks/BlueBook/Bluebook.pdf). 

The VM supports either a Qt or an SDL2 based human interface.

This is work in progress!
The version compiled with micc for x86 seems to work. 
There are still issues with the interpreter and C generator.
